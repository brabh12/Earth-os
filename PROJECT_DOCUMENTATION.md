# Earth OS: Planetary Intelligence Platform

## 🌍 Project Overview
Earth OS is a high-performance "Planetary Operating System" designed to monitor, analyze, and mitigate environmental crises at a global scale. It combines real-world satellite telemetry with advanced Large Language Models (LLMs) to create an autonomous feedback loop for environmental intervention.

---

## 🛠 How It Works
The platform operates on a three-tier intelligence architecture:

1.  **Observation Layer**: Uses WMS (Web Map Service) protocols to pull multi-spectral imagery directly from orbiting satellites.
2.  **Reasoning Layer**: An AI core powered by **GPT-4o-mini** (via OpenRouter) that interprets raw data, creates intervention strategies, and estimates environmental impact.
3.  **Action Layer**: A mission-based framework where "Volunteers" and "Organizations" can coordinate to solve the issues identified by the AI.

---

## 🚀 Key Features

### 1. Planetary Map (Digital Twin)
A high-fidelity 3D/2D mapping interface using Leaflet and React-Leaflet. It features:
*   **Earth OS Dark Mode**: A custom digital twin aesthetic for high-contrast data visualization.
*   **Copernicus Sentinel-2 Layer**: Real-time integration with European Space Agency imagery.

### 2. Autonomous Anomaly Registry
The system scans the globe for five distinct threat types:
*   **Deforestation**: Rapid loss of tree canopy.
*   **Desertification**: Soil degradation in arid regions.
*   **Water Scarcity**: Critically low aquifer or reservoir levels.
*   **Pollution**: Major chemical or plastic concentrations.
*   **Climate Anomalies**: Unexplained heat spikes or atmospheric shifts.

### 3. AI Command Console
When a threat is selected, the **Earth Intelligence Core** (via OpenRouter) performs a "Deep Scan" to:
*   Generate a **3-stage Intervention Plan**.
*   Calculate a **Confidence Score** for the detection.
*   Estimate **Resource Requirements** (e.g., specific technology or personnel).
*   Project **CO2 Sequestration Gains**.

### 4. Verification Portal
A crowdsourced "Proof of Impact" system where field reports are submitted and verified by the AI to ensure mission success.

## 🔗 Real-World Connectivity

### 📡 Satellite Integration
The project is connected to the **Sentinel Hub OGC services**. Every time you toggle the "Copernicus Sentinel-2" layer, the app makes a live WMS request to the Sentinel-2 satellite constellation archives. This provides imagery with a 5-day revisit time, meaning you are looking at the actual state of the planet within the last week.

### 🧠 OpenRouter AI Link
The "Intelligence" isn't a pre-written script. It is a live connection to the **OpenRouter API**. Each mission plan is generated on-the-fly based on the specific coordinates and type of threat, allowing the system to react to unique environmental contexts.

### 🗄️ Supabase Backend
All profiles, verified reports, and mission states are stored in a live **Supabase PostgreSQL** database. This allows for real-time synchronization between different users across the globe.
